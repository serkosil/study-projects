# Проект "Исследование объявлений о продаже квартир"

__Инструменты__: предобработка данных, Python, Pandas, Matplotlib, визуализация данных, исследовательский анализ

__Задача__: используя данные сервиса Яндекс.Недвижимость, определить рыночную стоимость объектов недвижимости и типичные параметры квартир.

На основе данных сервиса Яндекс.Недвижимость определена рыночная стоимость объектов недвижимости разного типа, типичные параметры квартир, в зависимости от удалённости от центра. Проведена предобработка данных. Добавлены новые данные. Построены гистограммы, боксплоты, диаграммы рассеивания.
